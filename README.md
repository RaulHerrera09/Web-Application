## Web-Application 

Esta aplicación web permite analizar datos de anuncios de venta de vehículos de forma interactiva. Utiliza Streamlit para crear la interfaz y Plotly Express para generar gráficos dinámicos. El usuario puede cargar el dataset, explorar visualmente la información y generar gráficos como histogramas o diagramas de dispersión para observar tendencias, por ejemplo, cómo varía el precio según el kilometraje. En resumen, facilita la comprensión de los datos de autos en venta mediante visualizaciones intuitivas y fáciles de usar desde un navegador web.
